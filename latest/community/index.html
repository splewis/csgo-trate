Community & Support

Discord

A Discord server is available for general support for Get5 (and other splewis plugins). If you have suggestions or feedback you are welcome to join.

Join Discord

No such thing as a free lunch

The Discord server is not a place to ask for general support with regards to setting up a CS:GO server or getting help with other plugins that we have nothing to do with.

GitHub

If you can code, you can help us over at GitHub. We're always looking for people to fix issues or help test new features. We provide nightly releases of Get5 before official releases.

Localization

If you speak a language that Get5 has not been translated into, or if you want to fix poor grammar or inconsistencies in current translations, you can join the Discord and let us know what's supposed to be changed. If you want to contribute yourself, take a look at the translations section.
